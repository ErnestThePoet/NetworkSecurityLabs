# Lab1
## FileBridge
基于TCP套接字和自定义应用层协议的Linux C/S结构双向文件传输应用，自动对传输数据进行SHA256验证，服务端支持多线程  

使用了来自[crypto-algorithms](https://github.com/B-Con/crypto-algorithms)的SHA256库
